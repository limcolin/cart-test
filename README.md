## Environment
**Requires:**
Node v12.1.0
NPM 6.9.0

## Running the development server
1.  ```sh
    npm install
    ```

2.  ```sh
    npm run dev
    ```

3.  View in the browser: http://localhost:8000

## Testing
1.  ```sh
    npm test
    ```

test file location in: src/templates/__tests__/cart-page.js
